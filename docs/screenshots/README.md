# Capturas de pantalla — Banco Pichincha Login

Coloca aquí las capturas del emulador o dispositivo físico para documentar el diseño en GitHub.

## Archivos sugeridos

| Archivo | Descripción |
|---------|-------------|
| `login_pantalla_completa.png` | Vista completa de la pantalla de inicio de sesión |
| `login_emulador_pixel.png` | Misma pantalla en emulador (p. ej. Pixel 6) |
| `login_scroll.png` | *(Opcional)* Comprobación de scroll en pantalla pequeña |

## Cómo tomar las capturas

1. Ejecuta la app en Android Studio (**Run ▶**) con un emulador o dispositivo conectado.
2. Abre la pantalla principal (login).
3. En el emulador: botón de cámara en la barra lateral, o `Ctrl + S` (Windows) / `Cmd + S` (macOS).
4. En dispositivo físico: combinación de botones de captura del fabricante.
5. Guarda los PNG en esta carpeta (`docs/screenshots/`).
6. Actualiza el README principal del repositorio insertando las imágenes:

```markdown
![Pantalla de login](docs/screenshots/login_pantalla_completa.png)
```
